# Capital Gains Tax Exercise

## Scenario
John is an investor with Apple stocks. Each Apple stock is currently valued at $135.  
He bought 20 stocks at $60 each, then acquired an additional 10 at $145 each.  
He sold 8 stocks at $120 each.  

This project calculates:
- Portfolio value
- Average purchase price
- Capital gains if sold now
- Tax implications in France (30%) and Belgium (first $500 exempt, 40% thereafter)

## How to Run
```bash
python capital_gains.py
